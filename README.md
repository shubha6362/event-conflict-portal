# Event Conflict Portal

A full-stack MERN application for managing events and venue bookings while preventing scheduling conflicts.

## Features
- Role-based authentication
- Event scheduling and management
- Venue booking system
- Conflict detection for overlapping events
- Search and filtering

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Installation

```bash
git clone https://github.com/shubha6362/event-conflict-portal.git
cd event-conflict-portal
npm install
npm start
```
